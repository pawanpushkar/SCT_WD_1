# 🌐 Responsive Landing Page with Interactive Navigation

## 📌 Project Overview

This project is part of the **SkillCraft Technology Internship – Task 01**.

The objective was to design and develop a **responsive landing page** with an **interactive navigation menu** that changes its style dynamically based on user interaction.

---

## 🎯 Features

* ✅ Fixed navigation bar (always visible on top)
* ✅ Changes color/style on scroll
* ✅ Hover effects on menu items
* ✅ Smooth transitions and animations
* ✅ Clean and responsive layout

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling, animations, responsiveness
* **JavaScript** – Scroll event handling and interactivity

---

## ⚙️ How It Works

* The navigation bar is fixed using `position: fixed`
* JavaScript listens to the **scroll event**
* When user scrolls, a class (`.scrolled`) is added
* CSS applies new styles like background color and shadow
* Hover effects are handled using CSS `:hover`
